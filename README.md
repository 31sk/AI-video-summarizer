# AI-video-summarizer
The AI Video Summarizer automatically extracts audio from any video, converts it to text using Automatic Speech Recognition (ASR), and generates a human-like summary using the BART transformer model. It is designed to help students, researchers, editors, and content creators get fast text summaries of long videos such as lectures, tutorials, podcasts, documentaries, and meetings.
